 ## 安装vue
- CND 的方式
 ```
<script src="https://cdn.jsdelivr.net/npm/vue@2.5.16/dist/vue.js"></script>
 ```
- npm 安装 node packge manager
```
 npm init
```
> 初始化会产生一个pacakge.json 的文件，这个文件会描述项目的依赖,不能有大写，中文，特殊字符，不要和安装包的名字相同，嗯就是这样。
